# `@compra-car/core`

Núcleo puro do Compra Car. Contém entidades, value objects, portas de repositório, erros e casos
de uso centrais do MVP, incluindo normalização, validação e criação administrativa de veículos.

O pacote não depende de Supabase, banco de dados, Next.js ou apresentação. Execute seus testes com `pnpm --filter @compra-car/core test`.
